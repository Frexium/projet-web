My Anime List

My Anime List est un site web qui vous permet de suivre vos animes préférés et de les noter. Vous pouvez aussi consulter les statistiques de votre liste d'animes.

Comment ça marche?

Vous pouvez ajouter des animes à votre liste en cliquant sur le bouton "Liste Animes" dans le menu. Vous pouvez ensuite noter les animes que vous avez déjà vus et consulter les statistiques de votre liste d'animes.

Comment ajouter un anime?

Pour ajouter un anime à votre liste, vous devez cliquer sur le bouton "Liste Animes" dans le menu. Vous pouvez ensuite cliquer sur le bouton "Ajouter un anime" et remplir le formulaire. Vous pouvez aussi rechercher un anime dans la barre de recherche.

Comment noter un anime?

Pour noter un anime, vous devez cliquer sur le bouton "Liste Animes" dans le menu. Vous pouvez ensuite cliquer sur le bouton "Noter un anime" et remplir le formulaire. Vous pouvez aussi rechercher un anime dans la barre de recherche.

Comment consulter les statistiques de ma liste d'animes?

Pour consulter les statistiques de votre liste d'animes, vous devez cliquer sur le bouton "Statistiques" dans le menu. Vous pouvez ensuite consulter les statistiques de votre liste d'animes.
